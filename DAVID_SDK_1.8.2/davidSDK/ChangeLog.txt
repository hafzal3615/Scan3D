 2015-09-04: What's New in DAVID SDK Version 1.8.0
--------------------------------------------------------------------
New features:
+ Vertices can now have Vertex Qualities [-1,1]. For now, used only in Fusion results
+ Fusion: Replaced parameter "bool closeHoles" by "float holeSizeThresRel", to define a hole size threshold, closing small holes and keeping larger holes open 
+ Fusion: Set Vertex Qualities: 1 for all "original" vertices. Values in [-1,0] for "closed holes", depending on each hole's size
+ IShapeFusion: New function GetVertexQualities
+ IShapeFusion: New function InvertTriangleOrientation
+ IShapeFusion: New function DuplicateMesh
